# 🚀 TaskFlow - Gestor de Tareas Inteligente

TaskFlow es una aplicación web dinámica diseñada para organizar tu día a día. Este proyecto forma parte de mi formación en desarrollo web, aplicando conceptos avanzados de **DOM**, **LocalStorage** y **CSS Flexbox**.

## ✨ Características
- **Persistencia de Datos:** Tus tareas no se borran al refrescar (LocalStorage).
- **Diseño Responsive:** Optimizado para móviles y escritorio.
- **Búsqueda en Tiempo Real:** Filtra tus tareas instantáneamente.
- **Gestión Intuitiva:** Añade y elimina tareas con un solo clic.

## 🛠️ Tecnologías utilizadas
- HTML5 Semántico
- CSS3 (Variables, Flexbox, Media Queries)
- JavaScript Vanilla (Manipulación del DOM, JSON API)
- Despliegue en **Vercel**

## 🚀 Instalación y uso
1. Clona el repositorio: `git clone https://github.com/TU_USUARIO/taskflow-project.git`
2. Abre `index.html` en tu navegador o usa Live Server en VS Code.
